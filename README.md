# 青少年学业倦怠水平评估系统

## 项目简介

这是一个专门为青少年设计的学业倦怠水平评估网站，通过科学的问卷评估、数据分析和科普知识，帮助青少年了解和改善自己的学习状态。

## 功能特色

### 📝 在线问卷评估
- 基于科学研究的10题学业倦怠问卷
- 逐步填写界面，减少认知负担
- 实时进度显示和结果计算
- 个人化的结果分析和建议

### 📊 数据可视化分析
- 多维度的数据分析仪表板
- 交互式图表展示群体趋势
- 年龄段、性别等筛选功能
- 详细的数据洞察和解读

### 📚 科普知识库
- 学业倦怠的定义和症状
- 影响因素和应对策略
- 交互式自测工具
- 专业支持资源推荐

### 🎨 现代化设计
- 响应式设计，适配各种设备
- 柔和的配色方案和优雅的动画
- 直观的用户界面和流畅的交互
- 专业的视觉设计

## 技术架构

### 前端技术栈
- **HTML5**: 语义化标签，确保可访问性
- **CSS3**: Tailwind CSS框架，自定义样式
- **JavaScript ES6+**: 原生JS实现所有交互
- **动画库**: Anime.js实现平滑动画效果
- **图表库**: ECharts.js创建数据可视化

### 核心功能
- 问卷逻辑和评分算法
- 本地数据存储和管理
- 动态图表生成和更新
- 响应式交互设计

## 页面结构

```
├── index.html          # 主页面 - 问卷填写和结果展示
├── data.html           # 数据分析页面 - 群体数据可视化
├── info.html           # 科普内容页面 - 知识和自测工具
├── main.js            # 主要JavaScript功能
└── resources/         # 图片资源文件夹
```

## 使用方法

### 本地运行
1. 将所有文件下载到本地目录
2. 使用Python启动本地服务器：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问 `http://localhost:8000`

### 功能使用
1. **问卷评估**：在主页完成10道题目，获得个人分析报告
2. **数据分析**：查看群体数据趋势和统计洞察
3. **科普知识**：学习相关知识，使用自测工具
