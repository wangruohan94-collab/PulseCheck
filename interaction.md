# 青少年学业倦怠网站交互设计

## 核心交互功能

### 1. 在线问卷系统
- **问卷填写界面**：基于问卷星的10题学业倦怠问卷
- **实时进度显示**：显示当前问题进度条
- **智能评分系统**：根据回答自动计算倦怠程度得分
- **结果分析**：提交后立即显示个人倦怠水平评估结果
- **数据存储**：将用户回答和结果存储在本地，用于后续数据可视化

### 2. 数据可视化仪表板
- **个人结果分析**：雷达图显示用户在各个维度的表现
- **群体数据对比**：柱状图展示不同倦怠程度的学生分布
- **趋势分析**：折线图显示不同时间段的数据变化
- **交互式筛选**：用户可以选择查看不同年龄段、性别的数据
- **动态图表**：使用ECharts实现丰富的数据可视化效果

### 3. 科普知识互动
- **倦怠程度自测**：基于问卷结果的简易评估工具
- **应对策略推荐**：根据用户得分推荐个性化的缓解方法
- **知识卡片**：点击展开详细的科普内容
- **资源库**：提供相关的学习资源和心理支持信息

### 4. 用户体验设计
- **渐进式问卷**：一次显示一个问题，减少认知负担
- **实时反馈**：每完成一个问题给予视觉反馈
- **结果分享**：用户可以生成个人结果报告图分享
- **数据导出**：支持将个人数据导出为PDF报告

## 交互流程
1. 用户进入网站 → 查看问卷介绍 → 开始填写问卷
2. 逐题回答 → 实时显示进度 → 完成后提交
3. 显示个人结果 → 查看详细分析 → 浏览科普内容
4. 查看群体数据 → 对比分析 → 获取应对建议

## 技术实现
- 使用本地存储保存用户数据
- ECharts.js实现数据可视化
- Anime.js添加平滑的交互动画
- 响应式设计适配不同设备
